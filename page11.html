Code Combat

THIS EVENT IS OPEN FOR ALL.
"If I have seen the future it is only by standing on the shoulders of giants.."
EVENT TYPE :
Team
REGISTRATION FEES :
 100 Per Team


  •   Game play: Each team consists of maximum two members
  • Languages Allowed : C,C++,Java,Python
  • The Event is divided into 3 rounds.
  • Preliminary round consists of 20 MCQs and 3 Problem statements along with sample input and desired output.
  •  For each correct answer 2 marks will be awarded.
  •  For every wrong answer 1 mark will be deducted
  •  For unanswered questions no marks will be awarded or deducted.
  • Marking scheme for Problem statement (60 marks):
  •  Marking scheme for Problem statement (60 marks): The code must work on various inputs. For every correct output to given test case, marks would be allocated.
  •  For unanswered questions no marks will be awarded or deducted.
  • Top teams will move to the next Round.
  • Time Allotted : 1hour 30minutes
  • Semi-Final round consists of 5 problem statements along with sample input and desired output.
  • Time Allotted (Semifinal ) : 1hour 30minutes
  • Final round consists of 3 problem statements along with sample input and desired output.
  • Time Allotted( Final ) : 2hours
  • In case of a draw, along with the score of each team, time taken by each team to complete the task will also be considered.
  • The team with highest score and lowest time wins the event.
  •  IN CASE OF ANY DISPUTES, THE DECISION OF EVENT CO-ORDINATOR WILL BE FINAL.
  • Contact : Biswajit Saha : 7003254546
All the above rules should be followed without argument with the tournament official, the violation of which may result in player or team disqualification, the decision to do the same lies in the hand of the tournament official present there, rules can be changed at any point of time as decided by the event co-ordinator present there.
